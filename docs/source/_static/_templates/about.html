We've all been there...

You're 4 hours into a fruitless debugging session and you want to go home - you don't have time for bourgeois luxeries like logging, type checking, or proper data visualization!
...But what if you did?
Would you write better code for your peers?
Would you publish more code online?
Would you finally be able to leave the optics lab for more time than it takes to walk downstairs and grab a stale poptart out of the vending machine?


We like to think so, and ImagePypelines can help!
